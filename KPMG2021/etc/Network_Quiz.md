### Problem
There are `...` collision domains and `...` domains.

### Answer
`17, 7`

## Quick Note
- `5` Router Port = `5` CD / `5` BD
- `5` Switch Port (No VLAN) = `5` CD / `1` BD
- `5` Hub Port = `1` CD / `1` BD
