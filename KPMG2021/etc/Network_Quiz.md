### Problem
There are `...` collision domains and `...` domains.

### Answer
`17, 7`

## Quick Note
- **Router** `5` Ports = `5` CD / `5` BD
- **Switch** `5` Ports (No VLAN) = `5` CD / `1` BD
- **Hub** `5` Ports = `1` CD / `1` BD
