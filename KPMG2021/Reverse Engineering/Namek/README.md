## Search String
1. Right-click, `Edit with Notepad++`, or use `Notepad` (or any editor of you choice)
2. `Ctrl + F` (Find)
    1. Search with RegEx : `KPMG\{(.*?)\}`
    2. or just search : `KPMG`
